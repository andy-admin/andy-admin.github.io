Мои требования к операционной системе на десктопе

Последнее изменение: 11/09/2019 16:19:28
  1. Должны быть предусмотрены горячие клавиши для быстрого запуска:
    • браузера;
    • почтового клиента;
    • файлового менеджера;
    • интернет-мессенджера;
    • простого текстового редактора (блокнота);
    • хорошо бы — калькулятора;
    • эмулятора терминала!!!
    • диалогового окна поиска приложений (чем меньше нажатий клавиш для запуска конкретного приложения, тем лучше);
  2. Должна быть возможность создавать несколько рабочих столов и переключаться между ними посредством горячих клавиш;
  3. На каждом рабочем столе на панели задач должны быть окна, открытые только на этом рабочем столе;
  4. При переключении между окнами они должны подсвечиваться;
  5. Для каждого окна (приложения) должна быть возможность запоминать раскладку клавиатуры;
  6. Переключаться между окнами одного приложения удобно сочетанием клавиш отличным от Alt+Tab, напр. Host+Tab;
  7. Должны работать сочетания клавиш для удобного размещения окон в областях рабочего стола: 1/2, 1/4, в любой угол, вверх, вниз, вправо, влево;
  8. Горячие клавиши управления окном: свернуть, развернуть максимально, развернуть на весь экран - маст хев;
  9. Уведомления о новых письмах или новых сообщениях должны быть видны на всех рабочих столах;
  10. Должна быть возможность без танцев с бубном подключать NFS/SMB-шары;
  11. Клиент удаленного доступа обязан работать и по RDP, и по VNC;
  12. На клавишу Host ничего не должно быть назначено по-умолчанию, чтобы пользоваться ею в удаленной windows-системе;
  13. Должна быть возможность настроить сочетания клавиш для смены раскладки клавиатуры (привет ubuntu 18.04 и невозможность использовать Ctrl+Shift!), самая годная для этого клавиша - CapsLock;
  14. Офис и PDF-ридер должны быть из коробки. Из офиса должна быть возможность экспорта в PDF;
  15. В репозиториях должны быть свежие версии нужных вам пакетов;
  16. Принтеры, съемные носители, блютуз-мыши и пр. устройства должны подключаться без проблем;
  17. ГУИ должен радовать глаз, а среда реже заставлять пользователя использовать мышь.

Мой выбор на сегодня — Xubuntu 18.04 LTS, предыдущий фаворит — Ubuntu 16.04 LTS (не удовлетворяет требрваниям из списка).

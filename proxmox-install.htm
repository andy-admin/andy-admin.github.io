Установка Proxmox VE на сервер Huawei 2288H V5

Последнее изменение: 08/06/2022 02:27:08

Проблема, кажется, появилась с выходом Proxmox VE версии 7.0. Установка не заканчивалась, но начиналась с ошибки "xinit: unable to connect to X server: Resource temporarily unavailable". Для справки, за графику на сервере ответственный:

 # lspci -s 02:00.0
 02:00.0 VGA compatible controller: Huawei Technologies Co., Ltd. Hi171x Series [iBMC
 Intelligent Management system chip w/VGA support] (rev 01)

Для запуска мастера установки в первой консоли потребуется выполнить следующе действия:

Создаем и перемещаем куда следует файл конфигурации X-сервера:

 # Xorg -configure
 # mv /root/xorg.conf.new /etc/X11/xorg.conf

Открываем файл на редактирование, ищем строку Driver и меняем значение "modesetting" на "fbdev" после чего сохраняем изменения и запускаем иксы сами:

 # nano /etc/X11/xorg.conf
 # startx

После окончания установки мастер, вероятно, не сможет перезагрузить сервер самостоятельно, поругавшись на display0. Придется отправить ему Ctrl+Alt+Del.
